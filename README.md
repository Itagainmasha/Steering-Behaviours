# Steering-Behaviours
simple hunter game in unity, practice steering behaviors

Для запуска скачать архив Build.rar, разархивировать и запустить .exe файл. Для движения игрока жать на WASD или на стрелочки, для выстрела - нажать пробел. Для рестарта игры - нажать на R.
Крестик - игрок, квадрат - волк, круг - заяц, треугольник - лани.
